# 변경 로그 (Changelog)

모든 주요 변경 사항이 이 파일에 기록됩니다.

## [1.0.0] - 2023-11-29

### 추가
- 포트폴리오 분석 및 시뮬레이션 기능 구현
- 자산 배분 전략 시뮬레이션 도구
- 다양한 포트폴리오 성과 지표 계산 (연평균 수익률, 샤프 지수, 최대낙폭, 변동성)

### 개선
- 모든 유틸리티 함수에 JSDoc 문서화 주석 추가
- 차트 렌더링 안정성 향상

## [0.9.0] - 2023-11-15

### 추가
- 리밸런싱 시뮬레이션 기능 구현
- 이동평균선 기반 자산 배분 전략 

### 개선
- 데이터 정규화 및 처리 로직 개선
- 포트폴리오 성과 계산 알고리즘 최적화

## [0.8.0] - 2023-11-01

### 추가
- 초기 프로젝트 구조 설정
- 기본 포트폴리오 분석 기능 구현
- 데이터 정규화 유틸리티 함수 추가
